## 📌 基本信息

### 项目简介
实现完整的用户管理功能，包括用户注册、登录、权限管理、信息查询和修改等核心功能。

### 技术栈
- **后端**: Node.js
- **数据库**: SQLite (liuxin.db) 或 独立用户数据库
- **认证**: JWT Token
- **加密**: bcrypt

### 项目目标
1. ✅ 实现用户注册功能（用户名、密码、邮箱）
2. ✅ 实现用户登录功能（JWT认证）
3. ✅ 实现用户信息查询功能
4. ✅ 实现用户信息修改功能
5. ✅ 实现用户权限管理功能
6. ✅ 实现用户列表查询功能
